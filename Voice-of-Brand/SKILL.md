# Marketing Claude Skills

A collection of Claude skills built for marketers, copywriters, and content creators. Each skill is a self contained instruction set that teaches Claude how to perform a specific writing or editing task consistently.

## What's inside

### anti-slop
Detects and flags AI sounding patterns in any piece of writing. Checks a draft against 21 known patterns (contrast addiction, thesaurus abuse, emoji explosion, generic case studies, and more) and returns a structured report with the exact lines to fix and how to fix them.

### copywriting-principles
Applies and audits copy against 22 core copywriting principles covering clarity, specificity, tone, and persuasion psychology. Includes a dedicated check for whether every ask in the copy is backed by a stated reason and social proof, the two psychological levers that make a request land harder.

### voice-founder
Extracts an individual founder's authentic voice from raw material like meeting recordings, journal entries, and organic posts, then writes new content that sounds like that person actually talks.

### voice-brand
Same approach as voice-founder, but built for a company or brand voice rather than an individual, so a team can write consistently on brand regardless of who's holding the pen.

## How to use a skill

1. Open the folder for the skill you want (for example `anti-slop`)
2. Copy the contents of `SKILL.md`
3. In Claude, create a Project and paste the skill content into the Project instructions, or paste it directly into a conversation
4. If the skill has a `references` folder, upload those supporting files to the same Project so Claude can read them when needed
