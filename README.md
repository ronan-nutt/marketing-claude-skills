My Go to Marketing Claude Skills

A collection of Claude skills built for performance marketers, copywriters, and content creators. Each skill is a self contained instruction set that teaches Claude how to perform a specific writing or editing task consistently, I use these EVERYDAY and they will be constatntly refined. 

What's inside (So far)

### anti-slop
Detects and flags AI sounding patterns in any piece of writing. Checks a draft against 21 known patterns (contrast addiction, thesaurus abuse, emoji explosion, generic case studies, and more) and returns a structured report with the exact lines to fix and how to fix them.

### copywriting-principles
 A reference skill covering core copywriting principles Claude applies when writing or reviewing marketing copy, ads, and sales content.

### voice-founder
Trains Claude on an individual founder's voice using their existing posts, organic content, meeting recordings, and journal entries, then writes new content that matches how that person actually talks and thinks.

### voice-brand
Same approach as voice-founder, but built for a company or brand voice rather than an individual, so Claude can write on-brand content across a team.

## How to use a skill

1. Open the folder for the skill you want (for example `anti-slop`)
2. Copy the contents of `SKILL.md`
3. In Claude, create a Project and paste the skill content into the Project instructions, or paste it directly into a conversation
4. If the skill has a `references` folder, upload those supporting files to the same Project so Claude can read them when needed
